Minimal kolom: ST_NUM (nomor rumah), ST_NAME (nama jalan).
Ambil HP dari folder HP/HOME. Jika tidak ada, ambil semua Point.
Tip: pilih jalan → pilih “Missing in Selected Street” → aktifkan Bulk Add.
Cara: pilih jalan → klik Start Bulk → sistem pilih missing pertama → kamu tinggal klik map berulang kali sampai selesai.
